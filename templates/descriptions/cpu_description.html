CPU Specifications

Processors are best understood by how many cores they have, how fast they run, and whether they include integrated graphics.

  • Core Count: The number of processing cores available for multitasking and parallel workloads.
  • Core Clock: The base operating speed of the CPU, usually shown in GHz.
  • Boost Clock: The maximum short-term speed the CPU can reach under thermal and power headroom.
  • TDP: A rough indicator of heat output and cooling demand.
  • Graphics: Whether the processor includes integrated graphics or requires a separate GPU.
  • SMT: Whether simultaneous multithreading is supported for handling more threads per core.
  • Microarchitecture: The CPU generation and design family, which affects efficiency and performance.
