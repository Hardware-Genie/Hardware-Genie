Memory Specifications

Memory kits are defined by capacity, speed, latency, and physical layout. Higher speed and lower latency can improve responsiveness, especially in bandwidth-sensitive workloads.

  • Modules: The number of sticks included in the kit.
  • Speed: The rated transfer rate, usually shown in MT/s.
  • CAS Latency: The number of cycles between a request and when the data is ready.
  • First Word Latency: The total real-world delay before the first data arrives.
  • Color: The heat spreader or aesthetic finish, useful for matching a build theme.
