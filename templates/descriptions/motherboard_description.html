Motherboard Specifications

Motherboards determine which processors and memory a system can use, along with how much expansion and I/O the build supports.

  • Max Memory: The highest RAM capacity supported by the board.
  • Memory Slots: The number of DIMM slots available for installing RAM.
  • Socket: The CPU socket type required for compatibility with a processor.
  • Form Factor: The physical board size, such as ATX or Micro-ATX, which affects case fit.
  • Color: The board's visual theme, often used when matching a build aesthetic.
